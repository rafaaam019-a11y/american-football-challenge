# 🏈 American Football Challenge

Projeto desenvolvido em Unity como atividade acadêmica do curso de Jogos Digitais da FATEC Americana.

## 🎮 Sobre o Jogo

American Football Challenge é um jogo 2D onde o jogador controla um atleta de futebol americano e deve lançar a bola em direção aos gols enquanto evita os defensores que tentam bloquear os arremessos.

O objetivo é marcar o máximo de pontos possível e superar seu próprio recorde.

---

## 🚀 Funcionalidades

✅ Menu Inicial

✅ Sistema de Pontuação

✅ Sistema de Recorde (PlayerPrefs)

✅ Sistema de Vidas

✅ Defensores com movimentação automática

✅ Colisão entre bola e defensores

✅ Tela de Game Over

✅ Interface com HUD de Pontos e Recorde

---

## 🎯 Controles

| Tecla | Ação |
|---------|---------|
| A / ← | Mover para esquerda |
| D / → | Mover para direita |
| Espaço | Arremessar bola |

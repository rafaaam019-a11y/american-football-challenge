## 📖 Sobre o Projeto

American Football Challenge é um jogo 2D desenvolvido em Unity utilizando a linguagem C#.

O projeto foi criado com o objetivo de aplicar conceitos de desenvolvimento de jogos, programação orientada a objetos, gerenciamento de cenas, física 2D e interface gráfica durante o curso de Jogos Digitais da FATEC Americana.

O jogador controla um atleta de futebol americano e deve lançar a bola em direção aos gols para marcar pontos, enquanto evita os defensores que tentam bloquear os arremessos.

---

## 🎮 Gameplay

### Objetivo

Marcar a maior quantidade de pontos possível acertando os gols.

### Mecânicas

- Movimentação horizontal do jogador
- Arremesso de bola
- Sistema de pontuação
- Sistema de recorde
- Sistema de vidas
- Defensores móveis
- Colisões entre bola e defensores
- Tela de Game Over

---

## 🕹️ Controles

| Tecla | Ação |
|---------|---------|
| A | Mover para esquerda |
| D | Mover para direita |
| ← | Mover para esquerda |
| → | Mover para direita |
| Espaço | Arremessar bola |
